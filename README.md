# 🤝 Pesona Baubau Digital Hub — Portal Mitra

Portal untuk mitra pariwisata: UMKM, Hotel/Homestay, dan Pokdarwis.

## 🚀 Tech Stack
- React + Vite
- Tailwind CSS
- React Router (routing)
- Zustand (state management)

## 💼 Jenis Pengguna
- 🏪 UMKM — kelola produk & lihat statistik
- 🏨 Hotel/Homestay — kelola kamar & reservasi
- 👥 Pokdarwis — kelola destinasi binaan & laporan

## 📁 Acuan Desain
File desain dari Google Stitch ada di `public/acuan-desain/` — buka screen.png untuk visual.

## 🏗 Cara Run
```bash
npm install
npm run dev
npm run build
```
